## Project details
**Project prompt:** [link to course webpage](https://www.nbi.dk/~petersen/Teaching/ML2026/FinalProject.html)

**Project title:** CNN Classifier for Audio Recognition trained on the FSD50k Data Set

**Project description:** Recognise simple audio clips by looking at spectrograms. Train on the FSD50K data set. Gain familiarity with the data set with the purpose of creating the best-performing CNN classifier possible with both single-label and multi-label models. Test various model enhancements and compare results between them, as well as comparing with results from the original FSD50k paper, and results from pre-trained models. If possible, try to apply the model to real-time audio.
